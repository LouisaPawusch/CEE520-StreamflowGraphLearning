# CEE520-StreamflowGraphLearning

USGS Streamflow data courtesy of the U.S. Geological Survey.
Precipitation data from https://cw3e.ucsd.edu
Gridded precipitation data are the National Stage IV Quantitative Precipitation Estimate (QPE) Product, provided by the NOAA/NCEP/EMC and PRISM daily precipitation estimate from PRISM Climate Group, Oregon State University.

Data made available through HydroFrame:
Defnet et al., (2024).  hf_hydrodata: A Python package for accessing hydrologic simulations and observations across the United States  Journal of Open Source Software, 9(99), 6623

For reproducing the downloading of the data, register on hydrogen (https://hydrogen.princeton.edu/signup?origin=hydroframe) and save the email and pin in an .env file in the root of the repository like this:

HF_EMAIL = *YOUR_EMAIL*

HF_PIN = *YOUR_PIN*
